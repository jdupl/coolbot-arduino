# LCD pinout

Connections from LCD headers to arduino


## Top left header

| 0 | ...| 6 (LCD E) | 7 (LCR RS)|
| ------------- | ------------- | ------------- | ------------- |
| N/C  | ... | D11  | D12 |

## Top right header

| 0 (LCD D7) | 1 (LCD 6) | 2 (LCD D5) | 3 (LCD D4) |
| ------------- | ------------- | ------------- | ------------- |
| D2 | D3 | D4 | D5 |

## Bottom left header

| 0 | 1 | 2 (5V) | 3 (GND) |
| ------------- | ------------- | ------------- | ------------- |
| NC | NC | 5V | GND |
